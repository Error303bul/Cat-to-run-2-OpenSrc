# Cat to run 2 OpenSrc

Этот репозиторий - сборник версий Cat to run 2, сделанной с помощью Replit.

## Игра

Чтобы поиграть в игру перейдите по этой ссылке: https://Cat-to-run-2.error303bulh.repl.co
